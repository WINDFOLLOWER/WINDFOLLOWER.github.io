持续摸鱼中
本篇主要讲的是LBS,我采用的是高德地图的SDK,我们今天要达到的目标主要是获取当地的经纬度和位置信息,完成这个就ok啦!
首先使用高德的定位服务需要注册为高德的开发者,网站:高德开发者网址
然后我们打开自己的个人中心,点击我的应用->创建新应用.

在这里我们需要填写自己的程序的各种信息,具体的安全码和PackageName获取方法上面都给你了,具体可以查看他们给你的教程.这里填写的相关的东西务一定不要出错,如果填错的话和你的应用信息不一致将不会传回位置信息.

配置工程

在工程中配置高德SDK官方也有相应的教程:官方配置工程教程,我这里推荐的是使用Gradle集成SDK(都8012年了还手动导入jar包),打开下图工程中的build.gradle文件

在build.gradle文件的dependencies的括号内需要根据我们的需要引入对应的代码

SDK 引入代码
3D地图 implementation ‘com.amap.api:3dmap:latest.integration’
2D地图 implementation ‘com.amap.api:map2d:latest.integration’
导航 implementation ‘com.amap.api:navi-3dmap:latest.integration’
搜索 implementation ‘com.amap.api:search:latest.integration’
定位 implementation ‘com.amap.api:location:latest.integration’

这里与官方唯一的区别在于使用的是implementation而非compile,原因在于3.0以上的AS中compile被标为过时.具体可以参考这篇文章android gradle tools 3.X 中依赖，implement、api 指令

由于今天我们要使用的只有定位只需要加入表中最后一项就行,当然你全加了也没没啥问题.然后点击AS代码框上方的Sync Now蓝色字AS就会自动下载并导入相关的jar包,到此为止配置已完成
